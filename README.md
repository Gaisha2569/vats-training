## หน้าเว็บ

| ไฟล์ | คำอธิบาย |
|---|---|
| `index.html` | **หน้าแรก** — หน้ารวม มีปุ่มไปบทเรียนวิดีโอและระบบฝึกอบรม |
| `lessons.html` | หน้าบทเรียนวิดีโอ 6 หัวข้อ |

ไฟล์ที่ขึ้น repo รวมประมาณ **45 MB**

## วิธีอัปเดตวิดีโอ

1. แทนที่ไฟล์ใน `videos/` ด้วยไฟล์ใหม่ (ใช้ชื่อเดิม)
2. เปิด `lessons.html` แก้ค่า `const VER="1";` เป็นเลขถัดไป เช่น `"2"`
   (สำคัญ — ถ้าไม่แก้ ผู้ที่เคยเข้าชมจะยังเห็นวิดีโอตัวเก่าที่ค้างใน cache)
3. `git add . && git commit -m "อัปเดตวิดีโอ" && git push`
